### I. Nexus — Gen-9 Autonomous Intelligence  
**Hybrid (On-Device + Cloud Distributed)** — the flagship layer.

- **DreamForge Engine (Gen-7)**: Creative reasoning, simulation, abstract problem-solving

### II. Darshana — Gen-6 Cloud Cognitive AI  
A cloud-native reasoning layer for contextual continuity and human-centric interaction.  
- Handles long-term memory, nuanced conversational context, cloud-scale reasoning.  
- Accessible via API — usable across phones, desktops, and laptops.  

### III. Panda — Gen-5 Lightweight Edge AI (Android)  
Edge-optimized on-device intelligence for fast, offline automation.  
- Directly hooks into Android intent system for high-speed automation.  
- Acts as entry gateway: for complex tasks, delegates to Nexus via secure channel.
